
- **Body** sẽ chỉ ra và nhấn mạnh các đặc điểm chính (**key feature**) mà đã xác định trong bước lập dàn ý.

- Để phân tích xu hướng của biểu đồ, có một số loại từ/cụm từ phổ biến là:

  - <span class="hl-warning">Từ/cụm từ miêu tả xu hướng</span>

  - <span class="hl-warning">Tính từ/Trạng từ miêu tả xu hướng</span>

---

### Từ/Cụm từ miêu tả xu hướng

- Khi viết **BP**, người viết cần chỉ ra điểm đạt mốc cao nhất/thấp nhất hoặc các dữ liệu có sự tăng mạnh/giảm mạnh. Vì vậy, cùng tham khảo các từ/cụm từ miêu tả xu hướng:

- **Ngôn ngữ chỉ điểm cao nhất:**

  - <span class="hl-warning">reach a peak</span>
  - <span class="hl-warning">peak</span>
  - <span class="hl-warning">reach/hit its highest point</span>

- **Ngôn ngữ chỉ điểm thấp nhất:**

  - <span class="hl-warning">reach its bottom</span>
  - <span class="hl-warning">reach/hit its lowest point</span>
  - <span class="hl-warning">hit a trough</span>

- **Ngôn ngữ chỉ tăng mạnh/giảm mạnh hoặc tăng giảm bất thường:**

| Tăng mạnh | Giảm mạnh | Tăng giảm bất thường |
| --------- | --------- | -------------------- |
| double    | drop      | fluctuate            |
| jump      | fall      |                      |
| rocket    | halve     |                      |
| shoot up  |           |                      |
| soar      |           |                      |
| surge     |           |                      |


### Tính từ & Trạng từ miêu tả mức độ thay đổi

- Người viết sử dụng các từ/cụm từ sau để bổ trợ mức độ cho các động từ/danh từ chỉ xu hướng.

- **Trạng từ chỉ tăng/giảm mạnh:**

  - <span class="hl-warning">dramatically</span>
  - <span class="hl-warning">significantly</span>
  - <span class="hl-warning">considerably</span>

- **Trạng từ chỉ tăng/giảm nhẹ:**

  - <span class="hl-warning">marginally</span>
  - <span class="hl-warning">negligibly</span>

- **Trạng từ chỉ tốc độ tăng/giảm:**

  - <span class="hl-warning">rapidly</span>
  - <span class="hl-warning">sharply</span>
  - <span class="hl-warning">gradually</span>
  - <span class="hl-warning">steadily</span>

### Lưu ý:

- Với dạng bài **line graph/bar graph**, biểu đồ thường sẽ không đưa ra các con số cụ thể.

- Người viết sẽ phải nhìn biểu đồ và đưa ra con số ước lượng (**approximation**) cho các dữ liệu.

- Một số từ/cụm từ thường đi kèm các con số để đưa ra con số áng chừng:

  - <span class="hl-warning">about</span>
  - <span class="hl-warning">almost</span>
  - <span class="hl-warning">approximately</span>
  - <span class="hl-warning">around</span>
  - <span class="hl-warning">close to</span>
  - <span class="hl-warning">just under</span>
  - <span class="hl-warning">just over</span>
  - <span class="hl-warning">nearly</span>
  - <span class="hl-warning">roughly</span>
